Step-by-Step Installation

Get up and running in under five minutes. Follow these simple configuration steps.

Configure your Windows PC environment

Ensure you have Python 3.10+ installed and added to your system PATH environment variable.

PowerShell 
# Clone the repository
git clone https://github.com/contactjayclatty/JAYX.git
cd JAYX/pc_agent

# Install standard dependencies
python -m pip install -r requirements.txt

# Install the Flipper build tool
python -m pip install ufbt

Compile and Launch on Flipper Zero

Connect your Flipper Zero via USB. Ensure any terminal console tools, serial monitors, or **qFlipper** are fully closed before continuing, to avoid port locks.

PowerShell 
# Move to the app directory
cd ../apps/jayx

# Compile and launch the app automatically
ufbt launch
Pro Tip: First-time builds may take a minute while `ufbt` downloads the Flipper SDK for your device firmware.

Open Connection & Stream

Start JAYX on your device and launch the local python agent process on your PC.

  1. On the Flipper: Open **Apps** → **USB** → **JAYX**.
  2. Highlight **USB** and press the **OK** button. The screen will say USB · waiting...
  3. On your Windows PC, open PowerShell and run the agent:
PowerShell 
cd ../../pc_agent
python monitor.py --usb

Within seconds, JAYX will establish the link and stream live stats!

Enable FPS and Temperature stats

Out of the box, JAYX streams system specs, CPU load, and RAM usage. Use these third-party utilities to unlock the complete suite of hardware readings:

🎮 Frame Rate Tracking (FPS)

Download and start RivaTuner Statistics Server (RTSS). Ensure it runs in the background. The FPS will show up on the Game page when you focus a game window.

🌡️ CPU Temperatures

Download and run LibreHardwareMonitor. Keep it active in the background tray. Restart the python agent to hook the sensors.

⚡ GPU Performance

NVIDIA Graphics metrics are supported natively via drivers. No extra tools are required if Nvidia drivers are installed. Other GPU brands will show metrics as N/A.
