High-Fidelity Tracking

JAYX monitors all core aspects of your Windows system performance, designed specifically for the Flipper's crisp 128x64 display.

CPU & RAM Status

Track active CPU load percentages, thermal temperature readings (via LibreHardwareMonitor), and live RAM allocation.

NVIDIA NVML Support

Integrates directly with the NVIDIA Management Library to stream GPU core utilization, VRAM usage, and GPU operating temperatures.

RivaTuner Game FPS

Hook game processes through RTSS (RivaTuner Statistics Server) to showcase current frame rates and game name titles on screen.

Detailed Hardware Cards

Use WMI querying to stream hardware names, OS editions, RAM clock speeds, and model architectures onto scrollable pages.

Ultra Low-Latency

Custom binary protocol over standard USB Virtual COM (CDC). Sends packaged metric packets every second without causing Flipper OS latency.

Wireless Bluetooth (WIP)

Bluetooth metrics transport layer is currently under active development. Keep an eye on updates to stream metrics completely cable-free.

Step-by-Step Installation

Get up and running in under five minutes. Follow these simple configuration steps.

Configure your Windows PC environment

Ensure you have Python 3.10+ installed and added to your system PATH environment variable.

PowerShell 
# Clone the repository
git clone https://github.com/contactjayclatty/JAYX.git
cd JAYX/pc_agent

# Install standard dependencies
python -m pip install -r requirements.txt

# Install the Flipper build tool
python -m pip install ufbt

Compile and Launch on Flipper Zero

Connect your Flipper Zero via USB. Ensure any terminal console tools, serial monitors, or **qFlipper** are fully closed before continuing, to avoid port locks.

PowerShell 
# Move to the app directory
cd ../apps/jayx

# Compile and launch the app automatically
ufbt launch
Pro Tip: First-time builds may take a minute while `ufbt` downloads the Flipper SDK for your device firmware.

Open Connection & Stream

Start JAYX on your device and launch the local python agent process on your PC.

  1. On the Flipper: Open **Apps** → **USB** → **JAYX**.
  2. Highlight **USB** and press the **OK** button. The screen will say USB · waiting...
  3. On your Windows PC, open PowerShell and run the agent:
PowerShell 
cd ../../pc_agent
python monitor.py --usb

Within seconds, JAYX will establish the link and stream live stats!

Enable FPS and Temperature stats

Out of the box, JAYX streams system specs, CPU load, and RAM usage. Use these third-party utilities to unlock the complete suite of hardware readings:

🎮 Frame Rate Tracking (FPS)

Download and start RivaTuner Statistics Server (RTSS). Ensure it runs in the background. The FPS will show up on the Game page when you focus a game window.

🌡️ CPU Temperatures

Download and run LibreHardwareMonitor. Keep it active in the background tray. Restart the python agent to hook the sensors.

⚡ GPU Performance

NVIDIA Graphics metrics are supported natively via drivers. No extra tools are required if Nvidia drivers are installed. Other GPU brands will show metrics as N/A.

Troubleshooting

Encountered an issue? Here are the most common solutions to get your JAYX monitor running.

  • You have run python monitor.py --usb in your PowerShell console.
  • The Flipper is connected with a reliable USB data cable.
  • No other software (like qFlipper) is open and locking the device's COM port.

Another application has already opened a serial connection to your Flipper Zero. Before uploading with ufbt launch or starting the metrics script:

  • Close the **qFlipper** application completely.
  • Kill any orphaned python processes by running: Get-Process | Where-Object {$_.ProcessName -eq "python"} | Stop-Process in PowerShell.
  • Unplug and replug the Flipper Zero to reset its connection port.

These require third-party tools running in the background of your PC. Without them, JAYX will still run smoothly but won't show those metrics:

  • For **CPU Temp**: Start **LibreHardwareMonitor** and leave it running.
  • For **Game FPS**: Start **RivaTuner Statistics Server (RTSS)**, run your game, and ensure that the game window has focus.
  • For **GPU Stats**: Ensure you have an NVIDIA graphics card. Other graphics architectures (AMD/Intel) are not supported by NVML and will show N/A.

Your local SDK is out of date compared to the firmware currently installed on your Flipper. Run the following commands inside `apps/jayx` to sync:

ufbt update
ufbt launch
