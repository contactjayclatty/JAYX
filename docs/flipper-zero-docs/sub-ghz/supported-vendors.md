# Supported Sub-GHz Vendors

## Overview

Flipper Zero can work with radio remotes from various vendors. The documentation states: "if your radio remote is not supported, you can help add the remote to the list of supported devices."

Users can share information about unsupported remotes on the community forum for analysis and potential addition to supported protocols.

## Protocol Legend

The page categorizes remotes by encryption type:

- **%unlocked% static**: Not encrypted protocols that Flipper Zero can decode, save, and playback
- **%locked% dynamic**: Encrypted protocols where Flipper Zero can decode signals, but the save function is disabled for security reasons

## Supported Vendors

The documentation includes an extensive table listing vendors and devices organized by:
- Vendor name
- Specific device models
- Encryption protocol type (static/dynamic)
- Frequency and modulation (e.g., 433.92 AM, 315.00 AM)
- Proven compatibility status

Notable vendors include AirForce, Alligator, Allmatic, Alutech, Chamberlain, Came, Faac, Genie, Hormann, Linear, Marantec, Nice, Sommer, and many others, covering a wide range of garage door openers, gate remotes, and similar radio-controlled devices.
