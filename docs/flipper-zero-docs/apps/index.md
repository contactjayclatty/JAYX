# Apps

## Overview

The Flipper Zero documentation page describes a community-driven app catalog that expands device functionality. Users can access applications through the Flipper Mobile App or Flipper Lab (web-based interface supporting Chromium browsers with Web Serial API).

## Accessing the Apps Catalog

**Via Mobile:**
- Download Flipper Mobile App on iOS or Android
- Connect smartphone to Flipper Zero via Bluetooth
- Tap "Apps"

**Via Desktop:**
- Connect Flipper Zero to computer via USB-C
- Visit lab.flipper.net in browser
- Click "Apps" in left sidebar

## Browsing and Managing Apps

Users can:
- Sort apps by category
- Filter and search for specific applications
- Preview app details, size, and version information
- Access changelogs and developer repositories
- Report bugs or inappropriate content
- Hide apps (mobile only)

## Installation

To install an app:
1. Navigate to Apps catalog
2. Find desired application
3. Click/tap the install button
4. Wait for upload completion
5. Access installed app via Main Menu > Apps

## Troubleshooting Installation Failures

Common issues include:
- Device not connected to mobile/computer
- Insufficient microSD card space
- Outdated firmware version
- Outdated app (contact developer via GitHub)

## App Management

Users can update or delete installed applications through the Apps section by locating the app under "Installed" and selecting the appropriate action.

## Developer Submissions

Developers can contribute applications by following the contribution guide on GitHub's flipper application catalog repository.
