# Remotes in Flipper Mobile App

## Overview

The Flipper Mobile App enables users to control various infrared devices through a remotes library feature. According to the documentation, this functionality supports "a wide range of devices, such as tvs, projectors, fans, and others" by allowing users to select the appropriate remote and control devices via their phone screen.

## Key Features

**Advantages over universal remotes:**
The mobile app approach differs from universal remote functionality because it "lets you pick the right remote once and save it for future use, which is much faster." It also "offers more buttons, supports a wider range of devices, and lets you control additional parameters."

**Supported devices include:**
- TV
- Projector
- TV box
- Fan
- Air purifier
- DVD player
- Camera
- AV receiver

## How to Find and Select a Remote

The process involves:

1. Opening the Flipper Mobile App and navigating to Main Menu > Tools > Remotes Library
2. Selecting your device type
3. Choosing your device's brand
4. Testing signals by pointing the Flipper Zero at your device and tapping buttons
5. Confirming device response to identify the correct remote
6. Saving the remote with an optional custom name

## Usage Methods

Users can operate saved IR remotes in two ways:

1. **Using both the mobile app and Flipper Zero** - Control via the phone's virtual remote interface
2. **Using only Flipper Zero** - Navigate saved signals directly on the device using navigation buttons

The IR remote database is open source and available on GitHub, with community contributions welcomed.
