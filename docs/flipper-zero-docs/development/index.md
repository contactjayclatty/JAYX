# Development

## Overview

The Flipper Zero Documentation's Development section serves as a hub for developers looking to work with the device. The page directs developers to comprehensive resources hosted on the main developer portal.

## Key Resources

The documentation highlights several essential areas for developers:

**Developer Documentation**
- Core developer docs covering firmware development, plugins, and file format descriptions

**Hardware & Connectivity**
- Wi-Fi Developer Board guides addressing connection, setup, and debugging procedures
- Technical specifications, schematics, and hardware drawings

**Programming & Scripting**
- "JavaScript on Flipper Zero" - featuring an MJS-based scripting engine with data types and built-in methods
- System programming resources for understanding firmware internals, unit tests, and API details

**Application Development**
- Custom app creation guidance including app manifests, examples, and the apps catalog
- File format descriptions organized by application type (iButton, IR, RFID, NFC, Sub-GHz, TAR)

**Developer Tools**
- Flipper Build Tool (FBT)
- VS Code workspace configuration
- Hardware and software tooling resources

**Additional Documentation**
- Command-line interface access and available commands
- Sub-1 GHz radio communication via CLI
- Driver troubleshooting for Windows systems

The section emphasizes that developers should consult the external developer documentation portal for detailed technical information across all these domains.
