# First Start - Flipper Zero Documentation

## Powering On

Press and hold the back button for 3 seconds to power on your Flipper Zero device.

## Rebooting

If your device freezes, hold the left and back buttons together for 5 seconds to reboot. The device supports rebooting into different modes, including recovery mode.

## Inserting a MicroSD Card

A microSD card is required for operation. The card stores keys, remotes, databases, and other data. While the device supports cards up to 256 GB, "a regular 16–32 GB microSD card is sufficient to store all the necessary data." Higher capacity cards may take longer to recognize.

## Updating the Firmware

Firmware updates are frequent and recommended. You can update via:

- **Flipper Mobile App** (iOS/Android) — Updates over Bluetooth, takes approximately 10 minutes
- **qFlipper** (desktop application) — Updates via USB cable, available for Windows, macOS, and Linux

### qFlipper Installation

Download the installer for your OS and run it. On Linux, the AppImage format requires making the file executable before use.

## Connecting Your Phone

1. Enable Bluetooth on both devices
2. Go to Main Menu > Settings > Bluetooth on the device
3. Set Bluetooth to On
4. In the mobile app, tap connect next to the detected device
5. Enter the pairing code and confirm

## Customizing System Preferences

Access Main Menu > Settings > System to adjust handedness, units of measurement, and time/date formats.
