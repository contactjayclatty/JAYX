# Desktop - Flipper Zero Documentation

## Desktop

The desktop functions as the digital pet's home environment, displaying the dolphin pet's status and emotional state. It features a comprehensive status bar at the top with various indicators showing device information.

## Status Bar Indicators

The status bar displays the following icons:

- **Battery fully charged** - device power status
- **Battery charging** - power connection active
- **Technical issue** - reboot required if persistent
- **MicroSD card mounted** - normal operation
- **MicroSD card mounting failed** - requires troubleshooting
- **Bluetooth Low Energy (BLE) active** - wireless connectivity enabled
- **BLE connected** - paired to another device
- **Device locked** - controls disabled
- **Device muted** - vibration and sound disabled
- **Connected to computer** - via qFlipper or Flipper Lab
- **Dummy mode active** - gaming device mode enabled
- **External module connected** - via expansion module protocol
- **BLE beacon mode** - broadcasting FindMy network signal
- **Background app running** - application active in background

## Navigation and Shortcuts

### Main Menu
Press the OK button to access features, settings, and apps including:
- Sub-GHz
- 125 kHz RFID
- NFC
- Infrared
- GPIO
- iButton
- Bad USB
- U2F
- Apps
- Settings

### Lock Menu
Press UP to access locking options:
- **Lock** - secures controls; press BACK three times to unlock
- **Lock with PIN** - password protection available in Settings
- **Mute** - disables sound and vibration
- **Dummy Mode** - restricts most functions for gaming

### Passport
View the digital pet's profile including name, appearance, mood, level, and progress.

### Archive
Access and manage saved data organized by category: Sub-GHz, RFID, NFC, Infrared, iButton, Bad USB, U2F, and Apps.

### Device Info
Press and hold DOWN to view firmware and hardware information.

### Quick Access Apps
Customize pad buttons (LEFT, RIGHT, DOWN, OK) to launch apps directly from desktop via Settings.
