# Reboot - Flipper Zero Documentation

## Reboot

The Flipper Zero documentation explains that firmware updates improve device performance, but occasionally the device may become unresponsive or freeze. The guide covers three reboot methods to address these issues.

### Normal Reboot

**With Hotkeys**
If your device freezes and doesn't respond to button presses, you can "reboot the device by pressing and holding the left and back buttons for 5 seconds."

**In Settings**
Navigate to Main Menu > Settings > Power > Reboot, then select the reboot option and press right.

**Via Flipper Mobile App**
After connecting your phone to the device via Bluetooth, tap Options in the main tab, then select Reboot Flipper.

### Hard Reboot

If a normal reboot doesn't resolve freezing issues, a hard reboot can help. "Press and holding the back button for 30 seconds" to reset the device's power circuit.

### Recovery Mode

For persistent firmware issues affecting display or responsiveness, you can reboot to recovery mode and use the qFlipper desktop application to repair corrupted firmware. The documentation directs users to the firmware recovery section for detailed instructions.
