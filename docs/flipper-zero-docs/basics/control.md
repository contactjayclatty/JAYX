# Flipper Zero Controls Documentation

## Overview

The Flipper Zero features built-in buttons that enable navigation, app access, and customizable hotkeys. Users can also enable left-handed mode to adjust controls accordingly.

## Basic Button Functions

**Directional Navigation:**
- Up, Down, Left, Right buttons navigate menus and virtual keyboards

**Action Buttons:**
- OK button launches apps and confirms selections
- Back button exits apps and returns to previous screens

**Selection:**
When encountering selectable options, use Left and Right buttons to move between them.

## Left-Handed Mode

To activate left-handed controls:
1. Navigate to Main Menu > Settings > System
2. Set the "hand orient" option to "lefty"

The button layout adjusts accordingly after switching modes.

## Hotkeys

**Power On:** Press and hold the Back button when powered off

**Normal Reboot:** Press and hold Left + Back buttons for 5 seconds

**Hard Reboot:** Hold Back button for 30 seconds

**DFU Mode Reboot:** Press and hold Left + Back for 5 seconds, then release Back while continuing to hold Left until the blue LED activates

Additional reboot information and power-off instructions are available in related documentation sections.

## Desktop Shortcuts

While on the desktop, buttons provide quick access to apps and menus. The quick-access apps are customizable for user preference.
