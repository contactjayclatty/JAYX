# Adding 125 kHz Cards Manually

## Supported Protocols

You can manually add virtual cards using these protocols:

- EM Micro EM4100
- EM Micro EM4100/32
- EM Micro EM4100/16
- HID H10301
- IDTECK
- IDTECK Motorola
- Indala 26
- Kantech
- ioprox
- XSF
- AWID
- Fecava
- FDX-A
- ISO FDX-B
- Generic HID Prox
- Generic HID Ext
- Farpointe
- Pyramid
- Viking
- Jablotron
- Paradox
- PAC
- Stanley
- Keri
- Gallagher
- Honeywell
- Nexwatch Electra
- Securakey
- Guardall G-Prox II
- Noralsy

## How to Add 125 kHz RFID Cards

To add a virtual card:

1. Go to Main Menu > 125 kHz RFID > Add Manually
2. Select the protocol and press OK
3. Enter the card's data in hexadecimal format, then press Save
4. Name the card and press Save

## Editing 125 kHz RFID Cards

To edit existing card data:

1. Go to Main Menu > 125 kHz RFID > Saved
2. Select the card you wish to edit and press OK, then Edit
3. Enter new data in hexadecimal format and press Save

Added cards can be emulated or written to blank T5577 cards.
