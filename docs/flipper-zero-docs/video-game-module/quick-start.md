# Video Game Module - Quick Start Guide

## Getting Started with the Video Game Module

The guide helps users install and connect Flipper Zero's video game module to a television.

## Installing the Module

The video game module arrives with a silicone bumper designed to fit snugly on Flipper Zero. Users with an existing silicone case should remove the bumper before installation.

## Firmware Requirements

The module requires Flipper Zero firmware version 0.98.3 or later. Users must update their device before using the module.

### How to Update via Flipper Mobile App

The update process involves these steps:

1. Download the Flipper Mobile App to a smartphone
2. Connect the phone to Flipper Zero via Bluetooth
3. Activate Bluetooth on both devices
4. In Flipper Zero's main menu, navigate to Settings > Bluetooth
5. Open the mobile app and select connect
6. Enter the pairing code shown on the Flipper screen
7. Confirm pairing
8. Tap the update button in the app's main menu tab
9. Confirm the update action

The process takes approximately 10 minutes. After completion, Flipper Zero automatically detects and initializes the video game module.

## Connecting to Your TV

Once firmware is updated and the module is attached:

1. Connect the TV cable to the video out port on the module
2. Access TV settings and select the appropriate input source

### Troubleshooting

If "video game module not initialized" appears, the firmware update likely wasn't completed. Users should follow the update instructions to resolve this issue.

## Firmware Updates

Additional firmware updates for the module will be released periodically. Users can consult the dedicated firmware update documentation for maintenance instructions.
