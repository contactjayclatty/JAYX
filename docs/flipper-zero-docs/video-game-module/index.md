# Video Game Module

## Overview

The Video Game Module is an accessory for Flipper Zero built around the Raspberry Pi RP2040 microcontroller. It connects via GPIO connectors and works with or without the silicone case.

## Key Features

**Screen Mirroring**
The module enables users to display Flipper Zero games and applications on a larger television or monitor screen via DVI-D video output.

**Motion Tracking**
"This module comes with a 6 axis motion tracking sensor that enables gyro controls in games." The sensor is a TDK ICM-42688-P and supports applications like Air Arkanoid and Air Mouse when connected to a computer.

**Standalone Capability**
The module can function independently with custom firmware, featuring a 14-pin GPIO header for external device connections.

## Ports and Controls

- **Video Out Port**: Transmits DVI-D format video to external displays
- **USB-C Port**: Connected to the RP2040 for firmware flashing and custom applications
- **GPIO Breakout**: 14-pin header for external device connectivity
- **Boot Mode Buttons**: Paper clip-accessible buttons for UF2 flash mode and reset functions
