# Video Game Module Firmware Update

## Installing the Video Game Module Tool

To update your module firmware, you'll need the Video Game Module Tool app. The documentation instructs users to "open the flipper mobile app" and search for the tool in the apps section, then tap install.

## Updating Official Firmware

The process involves three main steps:

1. **Initial Setup**: Connect the video game module to your Flipper Zero, navigate to Main Menu > Apps > Tools > Video Game Module Tool

2. **Installation**: Select "install official firmware" and confirm by pressing the install button

3. **Completion**: Upon success, you'll receive a confirmation message

## When Updates Are Available

If a newer official firmware version exists, users should:
- Update the Video Game Module Tool app itself through the Flipper Mobile App
- Reconnect the module to their device
- Return to the app and select "install official firmware" again

The documentation also notes that users can "install custom firmware" by selecting a file option, with additional details available in the custom firmware section.
